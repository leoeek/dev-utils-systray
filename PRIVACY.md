Política de Privacidade - Dev Utils Systray
Última atualização: 01/12/2025

A sua privacidade é importante para nós. Esta política de privacidade explica como o Dev Utils Systray ("o Aplicativo") trata os seus dados.

1. Coleta e Uso de Dados
O Dev Utils Systray foi projetado para priorizar a privacidade e o processamento local.

1.1. Dados Locais
A grande maioria das funcionalidades do aplicativo é executada inteiramente no seu dispositivo (offline).

Dados de Entrada: Qualquer código, texto, JSON ou imagem que você processa utilizando as ferramentas do aplicativo (ex: formatação, conversão) permanece na memória local do seu computador e não é enviado para servidores externos.

Preferências: As configurações do usuário (como temas, ferramentas favoritas) são salvas localmente em seu dispositivo (geralmente em um arquivo JSON na pasta de dados do usuário do sistema operacional).

1.2. Atualizações Automáticas
Para garantir que você tenha a versão mais recente e segura, o aplicativo pode verificar periodicamente se há atualizações.

Serviço Utilizado: Utilizamos a infraestrutura do GitHub Releases para distribuir atualizações.

Dados Transmitidos: Ao verificar atualizações, informações técnicas básicas (como seu endereço IP e a versão do sistema operacional) podem ser visíveis para os servidores do GitHub/Microsoft, conforme necessário para estabelecer a conexão. O aplicativo em si não coleta nem armazena esses dados.

1.3. Analytics e Rastreamento
Nós não utilizamos ferramentas de análise de terceiros (como Google Analytics, Mixpanel, etc.) dentro do aplicativo. Não rastreamos como você utiliza as ferramentas individuais.

2. Permissões do Sistema
O aplicativo pode solicitar permissões específicas do sistema operacional para funcionar corretamente:

Acesso à Área de Transferência (Clipboard): Necessário para funções de "Colar" e "Copiar" resultados das ferramentas. O aplicativo só acessa a área de transferência quando você executa uma ação explícita.

Notificações: Utilizado para alertar sobre a conclusão de tarefas ou atualizações disponíveis.

3. Serviços de Terceiros
O aplicativo é de código aberto e hospedado no GitHub. O código fonte pode ser auditado a qualquer momento em: https://github.com/leoeek/dev-utils-systray.

Não compartilhamos seus dados com terceiros, exceto conforme inerente à infraestrutura de distribuição (GitHub), mencionada na seção 1.2.

4. Segurança
Embora o aplicativo processe dados localmente, recomendamos que você mantenha seu sistema operacional atualizado e não utilize ferramentas de terceiros em máquinas comprometidas.

5. Alterações nesta Política
Podemos atualizar nossa Política de Privacidade de tempos em tempos. Recomendamos que você revise esta página periodicamente para verificar quaisquer alterações. Notificaremos sobre quaisquer mudanças postando a nova Política de Privacidade nesta página.

6. Contato
Se você tiver dúvidas ou sugestões sobre nossa Política de Privacidade, não hesite em entrar em contato abrindo uma Issue no repositório oficial: https://github.com/leoeek/dev-utils-systray/issues
